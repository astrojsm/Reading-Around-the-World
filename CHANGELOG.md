# Changelog

All notable changes to this project will be documented in this file.

## [1.2.1] - 2026-04-12

### Improved
- Refined overall UI layout and text readability for a cleaner presentation

### Changed
- Removed external version module (`_version.py`)

## [1.2.0] - 2026-04-12

### Added
- Added completion stamp on the hero section when challenge reaches 200 countries
- Added completion stamp rendering to shared image exports when challenge is completed

### Improved
- Improved shared image title and stamp placement balance
- Updated PDF header/section background and text color for improved readability

## [1.1.4] - 2026-04-11

### Improved
- Split world map into 3 panels on mobile for better readability

## [1.1.0] - 2026-04-11

### Improved
- Enhanced mobile responsiveness across key layouts and components
- Improved small-screen usability for country selection and controls

## [1.0.0] - 2026-04-11

### Added
- Record and manage books by country
- World map visualization with Plotly
- Export data as CSV format
- Generate and download shareable images
- Generate PDF summaries
- Multilingual text support (Korean, Chinese, Arabic, etc.)
- Session-based data management

### Improved
- Responsive interface (desktop/mobile)
- Intuitive form input structure
- Color-coded continents
- Enhanced PDF section header visualization
- Added latitude/longitude coordinates for small countries
- Improved Arabic text PDF rendering
- Automatic font selection (Noto Sans family)
