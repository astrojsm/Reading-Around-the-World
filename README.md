## Reading Around the World  


[https://reading-around-the-world.streamlit.app](https://reading-around-the-world.streamlit.app/)  


Reading Around the World 챌린지의 진행 상황을 추적할 수 있는 인터랙티브 앱입니다.  
국가별로 읽은 책을 기록하고 독서 여정을 세계 지도 위에 시각화할 수 있게 도와줍니다.

- 국가별 독서 기록 입력
- 세계 지도 기반 시각화
- CSV 파일로 데이터 저장
- 공유용 이미지 생성 및 다운로드
- PDF 요약 파일 생성
  
- 현재 앱은 한국어만 지원합니다.
- 모든 데이터는 세션에만 저장되며 새로고침 시 유지되지 않습니다.

---

A simple interactive app to track your progress in the "Reading Around the World" challenge.  
This app allows you to record books by country and visualize your global reading journey on a world map.

- Record books by country
- Visualize reading progress on a world map
- Export data as CSV
- Generate a shareable image
- Generate a PDF summary
  
- This app currently supports Korean language only.
- All data is stored locally (session-based) and is not persisted after refresh.
